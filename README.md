# OICA-Global-Sales

# Description
This project is data analysis of the publicly available OICA sale records. The purpose of this project is to demonstrate an understanding of data cleaning and dataset optimization through R. This project reconfigures the current available dataset chrnologically, and with columnized country, year, and sales. The cleaned dataset is free of errors, anomalies, and is made to be easy to navigate and analyze trends and patterns or perform visualizations. 

# Source
This project utilized data made available by OICA (The International Organization of Motor Vehicle Manufacturers).
